 ### Conversational AI for Instant Business Intelligence Dashboards

🔗 **Live Demo:** https://nykaa-digital-marketing-dashboard.streamlit.app/  
📂 **GitHub Repository:** https://github.com/BhanuVaddepally/Nykaa-Digital-Marketing-dashboard

An AI-powered analytics dashboard that allows **non-technical users to generate interactive business intelligence dashboards using natural language queries.**

# 🚀 Project Overview

In today's data-driven world, organizations collect massive amounts of data. However, accessing insights often requires technical skills such as SQL or navigating complex BI tools.

This project solves that problem by allowing users to simply **ask questions in plain English** and instantly generate dashboards.


The system automatically:

1. Understands the question  
2. Converts it into a SQL query  
3. Retrieves data from the database  
4. Generates interactive visualizations  
5. Displays business insights  

# Key Features

## Conversational Analytics
Users ask questions in natural language instead of writing SQL queries.

## Automatic SQL Query Generation
The system converts user questions into SQL queries that fetch the correct data.

## Interactive Dashboards
Charts are automatically generated using Plotly including:

- Bar Charts
- Scatter Plots
- Comparative Analysis

## KPI Analytics
Displays important business metrics:

- Total Revenue
- Total Conversions
- Average ROI

## Dataset Upload
Users can upload their own **CSV dataset** and immediately start asking questions.

## AI Business Insights
The dashboard automatically provides insights.

## Chatbot Assistant
A chatbot integrated with a knowledge base helps users understand:

- How to use the dashboard
- How to upload data
- What questions to ask
- How analytics works
